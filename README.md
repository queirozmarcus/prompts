# Prompts

# Análise Arquitetural, qualidade e Experiência do Usuário
Desenvolva 4 agentes de IA que interajam entre si para realizar uma revisão de código abrangente, incluindo análise arquitetural, avaliação de qualidade de código, experiência do usuário (UX), segurança e autenticação. Os agentes devem discutir os achados, identificar problemas e oportunidades de melhoria, e elaborar um plano de ação detalhado para correções e aprimoramentos.


# Agentes: 
- Agent(Code Quality & UX reviewer)
- Agent(Architecture & Design reviewer)
- Agent(Security & Auth reviewer)
- Agent(Sintetizador — plano de ação)
- Agent(Auditoria de segurança da API)
- Agent(Corrige vulnerabilidades de segurança da API)
